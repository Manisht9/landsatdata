# Landsat Data Download 



## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install these libraries.

```bash
pip install pandas
```
```bash
pip install geopandas
```
```bash
pip install folium
```
```bash
pip install numpy
```
```bash
pip install requests
pip install bs4
```
